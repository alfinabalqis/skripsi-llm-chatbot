# Proses Top Up Saldo pada Portal Layanan Digital Signature Aggregator (DSA)

> Sumber: `Proses Top Up Saldo Pada Portal Layanan DSA.pptx` (15 slide)

## 1. Halaman History Saldo

Menu **Saldo** menampilkan data total saldo beserta tabel history saldo. Untuk memudahkan pencarian, pengguna dapat menggunakan fitur **filter** pada tabel history saldo.

## 2. Langkah-Langkah Top Up Saldo

1. **Buka menu Saldo**, lalu klik tombol **Top Up Balance**.
2. **Input organisasi dan nominal top up** — nominal minimal **Rp10.000**.
3. **Pilih metode pembayaran**, yang tersedia:
   - E-Wallet
   - QRIS
   - Virtual Account
   - Virtual Account Recurring

   Setelah itu, pilih **jenis** metode pembayarannya.
4. **Cek Tagihan** — sistem akan menampilkan **Rincian Top Up**, kemudian lanjutkan pembayaran.
5. **Klik Bayar** — pengguna diarahkan ke halaman **Cara Pembayaran**, lalu klik **Lanjutkan Pembayaran** untuk mendapatkan kode.
6. **Input kode pembayaran** pada halaman jenis metode pembayaran yang dipilih. Kode pembayaran dapat dilihat melalui **Show Test Account** (mode pengujian).
7. **Input PIN** metode pembayaran. PIN dapat dilihat melalui **Show Test PIN** (mode pengujian).
8. **Konfirmasi pembayaran** — halaman detail *Payment Method* akan tampil, lalu klik **PAY** untuk menyelesaikan pembayaran.
9. Setelah pembayaran berhasil, sistem menampilkan **halaman konfirmasi keberhasilan pembayaran**.

## 3. Verifikasi Setelah Pembayaran

1. Kembali ke halaman **Cara Pembayaran** di aplikasi DSA, lalu klik **Cek Status**.
2. Jika berhasil, akan muncul pop up **"Saldo Berhasil diperbarui"**.
3. Pastikan saldo bertambah dengan mengecek kembali halaman menu **Saldo**.

## 4. Menu History Transaksi

Halaman **History Transaksi** menampilkan total transaksi yang telah digunakan, meliputi:

- Signing
- Stamp
- Reactive
- OTP
- Registrasi
- Session Code

Data ditampilkan berdasarkan **provider** dan **organization**.

Tersedia fitur **filter** berdasarkan tanggal (*date*), *provider*, *saldo type*, dan *organization* untuk memudahkan pencarian transaksi secara lebih efisien dan efektif.
